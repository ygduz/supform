# API Reference (v1)

Base URL: `http://localhost:8000`. Interactive docs (Swagger) at `/docs`, generated by
FastAPI. All endpoints are under `/api/v1`.

## Auth

| Method | Path | Body | Notes |
|---|---|---|---|
| `POST` | `/auth/signup` | `{email, password, full_name?}` | Create account |
| `POST` | `/auth/login` | `{email, password}` | → `{access_token, refresh_token}` |

Send `Authorization: Bearer <access_token>` on authenticated routes.

## Projects

| Method | Path | Notes |
|---|---|---|
| `POST` | `/projects` | Create a workspace |
| `GET` | `/projects` | List your projects |

## Forms

| Method | Path | Notes |
|---|---|---|
| `POST` | `/forms` | Create a form `{project_id, content: FormSchema}` |
| `GET` | `/forms/{id}` | Get a form (with draft content) |
| `PUT` | `/forms/{id}` | Update the draft `{content: FormSchema}` |
| `POST` | `/forms/{id}/publish` | Freeze draft → new version |
| `GET` | `/forms/{id}/schema` | **Public** — published schema for the renderer |

## Submissions

| Method | Path | Notes |
|---|---|---|
| `POST` | `/forms/{id}/submissions` | **Public** — submit `{answers, metadata?}` |
| `GET` | `/forms/{id}/submissions` | List responses (auth) |

## Errors

All handled errors return:

```json
{ "error": { "code": "validation_error", "message": "…", "details": { } } }
```

`validation_error` (422) on submissions includes a per-field `details` map.
